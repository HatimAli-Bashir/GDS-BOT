## English

A comprehensive Telegram bot designed to automate and deliver digital marketing services, boosting social media engagement instantly and automatically.

### 🌟 Key Features
* **Full Automation:** Instant order processing and execution with zero manual intervention.
* **Flexible UI:** Simplified and smooth menu design using Inline Buttons for an optimal user experience.
* **High Security:** Built with strict isolation of sensitive environment variables (`BOT_TOKEN`) to ensure a secure production environment.
* **Continuous Uptime:** Hosted on a reliable cloud platform ensuring 24/7 availability.

### 🛠️ Tech Stack
* **Language:** Python 3.14
* **Core Library:** pyTelegramBotAPI (Telebot)
* **Hosting:** Render Cloud (Web Service)
* **WSGI Server:** Gunicorn

### 🚀 Setup & Installation (For Developers)
If you want to run this project locally or self-host it:

1. Clone the repository:
```bash
   git clone [https://github.com/HatimAli-Bashir/GDS-BOT.git](https://github.com/HatimAli-Bashir/GDS-BOT.git)
   
    2.  Install the required dependencies: 
bash
pip install-r requirements.txt

3.Set your environment variables  (BOT_TOKEN) and run the bot:

bash
python gds.py
